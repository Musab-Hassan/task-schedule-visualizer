# Processor Task Scheduling Visualizer

A simple Svelte web app for visualizing CPU task scheduling with different scheduling algorithms (Rate Monotonic, Earliest Deadline First, Least Laxity First).

![Overview](assets/overview.gif)

[![Check out the Live App](https://img.shields.io/badge/Check%20Out%20the%20Live%20App-blue?style=for-the-badge)](https://taskscheduler.sandbox.musabhassan.com/)

## Development

```sh
npm run dev
```

Start the development server at `http://localhost:5173`.

## Building

```sh
npm run build
npm run preview
```

Build for production and preview the optimized app.

## Testing

```sh
npm run check
npm run check:watch
```

Run type checking with Svelte and TypeScript.
